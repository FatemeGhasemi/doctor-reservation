## Db
### init
For create db tables just enter this command 

`npm run db:init`

### Seed 
If you want seed db with mock data :

`npm run db:seed`

### Drop & Seed (Recommended)

if you want to drop tables first and then seed with mock data:

`npm run db:drop:seed`
